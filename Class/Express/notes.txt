Routing in express js
    Routing refers to determining how an application respond to client request for specific url and http method.
    Components of routing : URL path, HTTP method, handler function

Middleware in Express js
    Middleware is a function that execute during the request before sending the final response.
    They have access to request, response and next function.
    Flow :
        1. Request is made to the server
        2. Middleware function is executed
        3. Middleware can modify request and response objects
        4. Middleware can end the request-response cycle or pass control to the next middleware function using next().
